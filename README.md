# HireMind 

HireMind is an AI-powered resume–job alignment analyzer.

The tool analyzes how well a resume matches a job description by extracting skills from both and identifying missing competencies.

## Planned Features
- Resume PDF parsing
- Skill extraction from resume and job description
- Resume–job match scoring
- Missing skill suggestions

## Tech Stack
Backend: FastAPI, Python  
NLP: spaCy 
Frontend: Next.js
